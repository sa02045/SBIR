# 맵 생성 시 부수 효과를 피하라

- 맵의 사본은 어떻게 생성할까?
- 부수효과를 피하려면 어떻게 해야할까?

```js
const copy = new Map([...map]);
```

1. 키-값 쌍이 담긴 배열을 전달하면 새로운 맵을 생성할 수 있다
2. 맵을 펼침 연산자로 키-값 목록을 만들 수 있다
3. `[...map]` 배열로 만들어 Map()의 인자로 넘겨준다

맵 두개를 합치려면

```js
new Map([...map1, ...map2]);
```

## 정리

1. `new Map([...map])`으로 맵의 사본을 만들어 부수효과를 피하자
